# TimeTV
Thử làm ứng dụng đồng hồ bằng Javascript (real time). Sử dụng API từ bên ngoài với sự kết hợp của thư viện Jquery.

Bài viết: http://tutran.net/thu-lam-dong-ho-bang-javascript.html
